必备实用软件(Windows)

发表于 2020-10-04 更新于 2020-10-09
记录下自己安装过的觉得好用的软件

实用工具

Windows Terminal

Windows 终端是一个面向命令行工具和 shell（如命令提示符、PowerShell 和适用于 Linux 的 Windows 子系统 (WSL)）用户的新式终端应用程序。 它的主要功能包括多个选项卡、窗格、Unicode 和 UTF-8 字符支持、GPU 加速文本呈现引擎，你还可用它来创建你自己的主题并自定义文本、颜色、背景和快捷方式。

不知道比系统自带的Powershell和cmd高了多少B格

Internet Download Manager

非常强大的下载工具，懂得都懂

µTorrent

μTorrent(由于μTorrent不便于书写常作uTorrent [1] ) 是一个小巧强劲，全功能，用C++编写，支持Windows、Mac OS X和GNU/Linux平台的免费BitTorrent客户端。支持UPnP，支持流行的 BT 扩展协议，磁力链接(Magnet Links)，IPv6，用户来源交换，DHT和uTP，以及RSS下载器等丰富特性。

虽然基本上都用网盘了，但有些资源还是得靠这个

AssetStudio

AssetStudio ，原名 UnityStudio是一款专门用来浏览，提取和导出unity导出的apk中的asset和assetbundles的工具。

支持的资源类型包括：

  • Texture2d
  • Sprite
  • AudioClip
  • Font
  • Mesh
  • TextAsset
  • Shader
  • MovieTexture
  • VideoClip
  • MonoBehavior
  • Animator

PotPlayer

win10的播放器太烂，听说这款很强

Everything

Everything 是一个文件名搜索工具。它小巧免费，支持中文，支持正则表达式，可以通过 HTTP 或 FTP 分享搜索结果。

搜索速度极快，全盘搜索，且体积非常小，只有几MB。

KrkrExtract

专门用来解包Krkr引擎的GalGame

MobaXterm

相比较Putty以及系统自带ssh，这款终端软件很漂亮，并且功能也足够强大，大概只有Xshell能够一战。

OBS Studio

OBS Studio 是一款用于实时流媒体和屏幕录制的软件，为高效捕获，合成，编码，记录和流传输视频内容而设计，支持所有流媒体平台。

Typora

用过不少Markdown编辑器，比如vnote，VScode，还有各种在线编辑器，最终还是选择Typora。

OpenArk64

 OpenArk是一款Windows平台上的开源Ark工具. Ark是Anti-Rootkit（对抗恶意程序）的简写, OpenArk目标成为逆向工程师、编程人员的工具，同时也能为那些希望清理恶意软件的用户服务。以后也将会支持更多功能和命令。

虽然功能非常强大，但我只用来看端口占用以及热键冲突。

除此之外，还有一些非常实用的功能

  • 修改hosts

  • 编码、算法、汇编工具

  • 实用工具

KeePass Password Safe

KeePass Password Safe 就是专门为了解决人类记不得众多密码的问题所产生的，它包含了一个强大的密码产生引擎与加密储存机能，能够提供一个安全的密码储存空间。

虽然UI不够漂亮，但是好歹是免费的

Proxifier

Proxifier是一款功能非常强大的socks5客户端，可以让不支持通过代理服务器工作的网络程序能通过HTTPS或SOCKS代理或代理链。

只要设置规则，就可以强制让其走代理。

Snipaste

简单但强大的截图工具，支持贴图功能

Fiddler Everywhere

Fiddler Everywhere是适用于任何浏览器，系统或平台的高性能，跨平台的Web调试代理。它是一个Web调试工具，它记录计算机和Internet之间的所有HTTP（S）通信，检查和编辑通信，发出请求以及处理传入和传出的数据。

界面比原版要漂亮不少

不过目前还有bug，以及一些功能还不到位

Bitwarden

Bitwarden 是一款开源的密码管理器，支持 Web、Chrome，很快将要发布 Firefox、Opera 以及 Edge 浏览器插件，拥有 iOS、Android 客户端，采用本地加密，云同步的方式。

最初我使用的是Keepass+Onedrive的模式来进行密码管理以及云同步，不过Keepass的界面实在是难看，挑来挑去，最终选择这款密码管理软件。

这款工具相比较Keepass，UI更好看，并且自带云同步

缺点就是无法在本地存储密码，密码全部放在服务器上，万一没网就尴尬了。

语雀

「语雀」是一个「专业的云端知识库」，孵化自 蚂蚁金服 ，是 体验科技 理念下的一款创新产品，已是 10 万阿里员工进行文档编写、知识沉淀的标配。

语雀诞生伊始，只是希望能给工程师提供一个好用的工具用来写技术文档，达成「用 Markdown 写文档」这个小目标。但在产品研发的过程中，我们发现其实身边的每个人、每个团队、每个组织都有很多知识，但一直以来缺少一个好用的工具让这些知识不只是留在每个人的大脑或电脑里，还可以被记录、分享和交流。

语雀，个人感觉比较像Gitbook，Markdown编辑体验也很不错，目前用户量还比较小，未来可期

不过作为博客来说，语雀在百度上的权重占的还是比较低的

Digikam

试了很多图像管理工具，Eagle、Billfish、Tropy，最终还是选择Digikam，先说说其他软件的优缺点。

  1. 首先要说的就是Eagle：

    优点：

    • UI很漂亮，真的很漂亮
    • 强大的标签功能
    • 非常方便的浏览器插件功能

    缺点：

    • 没有云同步
    • 因为数据存储的方式，文件量大的情况下，读取速度会很慢，并且更严重的是有可能会丢失文件，很要命

    软件确实是很好用，但是这种文件丢失，甚至导致硬盘损坏的问题还是太致命了，并且官方也没给出实质上的回答，有股避实就虚的味道。以下是官网原文：

    问题一：使用 Eagle 是否会伤硬盘？

    用过电脑的人都知道，任何硬件无论使用与否，都有最终寿命。 所以，这个问题更应被探讨的地方是「Eagle 是否有不当、不正常的读写硬盘，间接造成硬盘损害」。

  2. Billfish

    感觉基本上就是模仿Eagle做出来的东西，使用方法和整体ui非常相似，但是存储是采用的sqlite。

    优点：

    • UI还可以，但有些小细节没有做好，比如图标和文字的对齐之类的
    • 浏览器插件功能
    • 占用内存小

    缺点：

    • Beta阶段，bug还是不少
    • 虽然是有浏览器插件，但是不够强大

  3. Tropy

    优点：

    • 界面简洁，直观
    • 可以安装插件

    缺点：

    • 功能不够强大，当作单纯的图像分类工具来说很合适

综上，最后还是选择了Digikam，功能很强大，上面工具有的，他都有，最新版本还增加了神经网络人脸识别功能。

但是这UI实在是次了一点，如果能大改一下，基本上就是第一选择了。
