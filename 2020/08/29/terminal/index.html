前言

Powershell这体验着实是不行，乱码，颜色无法显示。是时候该换Windows Terminal了

安装

Github安装

下载页面Github，下载下来安装就可以了，值得注意的是，如果从 GitHub 安装，终端将不会自动更新为新版本。

Microsoft Store安装（推荐）

直接在开始菜单中打开Microsoft Store，搜索Windows Terminal，安装即可
